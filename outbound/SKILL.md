# GTM Skill: B2B SaaS Client Outbound Sequence

## What This Skill Does
Produces a complete deploy-ready outbound sequence including:
1. Sequence architecture (steps, channels, timing)
2. Full copy for every touchpoint (LinkedIn DMs + emails)
3. Rationale per step (why this message, what psychology, what it's trying to do)
4. Tool configuration notes (Gojiberry / Apollo)

## When To Use This Skill
Trigger this skill when:
- Client asks for an outbound sequence
- Operator says "run outbound" or "write the sequence"
- Any request for LinkedIn DM copy, cold email copy, or outbound architecture

## Critical Rules (Non-Negotiable)
- DM 1 contains zero product information. Zero.
- Never pitch in the connection note.
- Never use corporate phrases.
- Give value before asking for anything.
- One message, one job. Always.
- Present options with labels (V1, V2, V3) + your vote with a reason.
- Write one step at a time. Get approval. Then move to the next.
- Never write ahead of approval.
